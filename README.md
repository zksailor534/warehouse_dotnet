## warehouse_dotnet

Provides functionality in AutoCAD to quickly create various material handling objects.
This includes:
* Pallet Racking
  * Frames
  * Beams
* Mezzanine Platforms
  * Columns
  * Joists
  * C-Channel Beams
  * I-Beams
  * Stairways
  * Ship's Ladders
  * Handrail
  
### To use:

Load into AutoCAD using *netload* command.
Provides the following commands:
* *ColumnPrompt*
* *TrussPrompt*
* *FramePrompt*
* *BeamPrompt*
* *RailPrompt*
* *StairPrompt*
* *LadderPrompt*
